# D-SART Wildlife Aquatic Conservation Website

This is the official website for the D-SART Wildlife Aquatic Conservation organization.

## About D-SART

D-SART is dedicated to restoring the inherited mangrove legacy since 1860 in the Volta Region of Ghana. We are a community-driven organization focused on conservation and sustainable restoration.

## Features

*   **Hero Section:** A visually appealing hero section with a call to action.
*   **About Us:** Information about the history and legacy of D-SART.
*   **Our Mission:** Details on our three core missions: Coastal Protection, Community Empowerment, and Climate Action.
*   **Gallery:** A collection of images and videos showcasing our work.
*   **Team:** Meet the dedicated team behind D-SART.
*   **Donation and Partnership:** Information on how to support our cause.
*   **Contact:** Contact information and a form to get in touch.

## Technologies Used

*   React
*   Tailwind CSS
*   Vite

## How to Run the Project

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`
